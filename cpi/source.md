# CPI Motor Vehicle Insurance

- BLS factsheet: https://www.bls.gov/cpi/factsheets/motor-vehicle-insurance.htm
- BLS Public Data API signatures: https://www.bls.gov/developers/api_signature_v2.htm
- BLS CPI series used: `CUUR0000SA0` for all items and `CUUR0000SETE` for motor vehicle insurance
- Data fields used: annual CPI-U averages, 1960-2025
- BLS-sourced yearly CPI table for all items: https://www.officialdata.org/us-cpi
- BLS-sourced yearly motor vehicle insurance table: https://www.officialdata.org/Motor-vehicle-insurance/price-inflation/1970
- Supporting context on teen-risk pricing: https://www.iii.org/issue-update/background-on-teen-drivers
- Supporting context on teen-driver insurance: https://www.iii.org/article/auto-insurance-teen-drivers
- Supporting context on pricing factors: https://www.iii.org/article/what-determines-price-my-auto-insurance-policy
- Story rationale note: `cpi/cpi-motor-vehicle-insurance/why-this-matters.md`
- Note: `CUUR0000SETOO` appears to be a typo; the BLS motor vehicle insurance series is `CUUR0000SETE`.
